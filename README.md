# typescript-rest-api-codebase
A structural REST API codebase using Node.js (Restify), Typescript, and Mongoose.

## Usage

```
# Build the application
npm run build

# Install dependencies
npm install

# Run in development
npm run dev

# Run in production
npm start
```
